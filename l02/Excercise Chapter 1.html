Discussion

1a) The software rules the hardware. It states commands for the hardware to execute. Without software a computer can't operate. So hardware is the physical counterpart to software.

1b) An algorithm is like a recipe to solve a certain, very specific problem. A program combines several algorithms to solve a more complex problem.

1c) Programming language is a very precise language which is constructed to give computers commands without ambiguity. Natural language is not constructed in that technical way and therefore contains a lot of imprecisions which would confuse the computer.

1d) High level language is a language which can be used by humans to write commands. However, those commands need to be translated into the language of the hardware (or Machine Language) which is based on a binary system. The commands must be translated so the cpu can process the commands.

1e) An interpreter simulates a computer which can understand high level language. It creates a more fexible and interactive programming environment. On the other hand, the compiler translates the high level language into the machine language. Compiled programs are usually faster because once they are compiled no translation is needed anymore.

1f) Syntax determines the structure of a programing language, whereas semantics contains its meaning.

  1. CPU: The CPU is the brain of the computer which carries out simple task like adding numbers or checking if they are equal
  2. Main Memory or RAM: The Main Memory can be accessed directly by the CPU which makes it fast to safe or access information. However, after shutting down the device all information is lost.
  3. Secondary Memory: The Secondary Memory stores information permanent.
  4. Input Devices: The Input Devices are used to interact with the computer. To type commands or input information which then are proccessed.
  5. Output Devices: The Output Devices display the output of the computer.
